数学。SQRT1_2 是 JavaScript 中的一个属性，简单来说就是用来求 1/2 的平方根值，其值近似为 **0.707106** 。
就是说，

> √ （1/2） = 0.707106

**语法:**

```
Math.SQRT1_2;
```

**返回值:**它只是返回 1/2 的平方根值，其值约为 7071。
下面的例子说明了 JavaScript 中的 Math SQRT1_2 属性。

*   **示例:**这里简单地将 1/2 的平方根值显示为输出。

```

Output : 0.7071067811865476
```
