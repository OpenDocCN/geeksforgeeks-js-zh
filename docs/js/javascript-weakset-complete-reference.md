[JavaScript 中的 WeakSet](https://www.geeksforgeeks.org/javascript-weakset/) 用来存储对象的集合。它采用与集合相同的属性，即不存储副本。WeakSet 和 Set 的主要区别在于 WeakSet 是对象的集合，而不是某种特定类型的值。
