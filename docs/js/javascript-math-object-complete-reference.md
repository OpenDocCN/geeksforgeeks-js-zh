数学对象用于对数字进行数学运算。

**示例:**

```
<!DOCTYPE html>
<html>
    <body>
        <h2>GeeksforGeeks</h2>

        <script>
// Return PI value(3.141592653589793)
            document.write(Math.PI);
        </script>
    </body>
</html>
```
